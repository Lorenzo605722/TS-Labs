# 🧪 TypeScript Labs

Raccolta di script focalizzati sulla **robustezza del codice** e la **validazione dei dati**.

### 🚀 Featured: Data Processor & Validator
Uno scenario reale di una startup social dove i dati grezzi del backend vengono puliti e tipizzati.

**Tecnologie utilizzate:**
* **Type Guards**: per convalidare stringhe e numeri in ingresso.
* **Discriminated Unions**: per gestire utenti "Attivi" vs "Inattivi" in modo sicuro.
* **Async/Await**: gestione di fetch API reali.

*Obiettivo: Dimostrare come TypeScript possa prevenire bug in produzione tramite una tipizzazione rigorosa.*
